# IoT Devices

[← README](../README.md)

All devices currently on `192.168.xx.xx` (flat network — VLAN migration pending). Target network for IoT: `192.168.xx.xx` (ssid-iot SSID). See [network.md](network.md) for subnet layout and WiFi SSIDs, and [security.md](security.md) for the IoT lateral movement threat.

Zigbee devices connect via IKEA Dirigera hub (`192.168.xx.xx`) — no individual IPs.

> IPs and MACs for WiFi devices: see `_snapshots/network-hosts.yaml` (`dhcp_reservations` section, generated by `_scripts/generate-network-snapshot.sh`).
